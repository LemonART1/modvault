About this mod

ArchiveXL is a fundamental modding tool for Cyberpunk 2077, designed to load custom resources without modifying the game's original files. This capability prevents conflicts and allows multiple mods to expand existing resources harmoniously. It enables developers to implement custom entity factories, add or edit localization texts, override visual elements, and create new character customization options. The tool also supports custom attachment slots, world streaming blocks, journal entry management, and widget spawning. It requires Cyberpunk 2077 2.31, RED4ext 1.29.0+, and redscript 0.5.31+.
