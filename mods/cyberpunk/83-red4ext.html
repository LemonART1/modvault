About this mod

RED4ext is a fundamental library that extends REDengine 4, enabling modders to add new features, modify game behavior, and introduce custom scripting functions in Cyberpunk 2077. This library consists of a loader for managing plugins and an SDK containing reversed types and helpers for direct engine extension. It allows developers to integrate new scripting capabilities or invoke existing ones within their own plugins. This crucial framework for advanced mod development requires the latest Visual C++ Redistributable 2022 to be installed. It serves a similar purpose to tools like Script Hook V and Skyrim Script Extender, providing core functionality for complex modifications.
