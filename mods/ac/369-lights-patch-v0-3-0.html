About this mod

The Lights Patch v0.3.0 is a modification for Assetto Corsa that aims to improve the game's lighting effects. The latest update, preview361full, addresses an issue with rainbow colors on glass materials by using an internal noise map that matches the one in version 0.2.11. Additionally, the patch fixes a problem with brake discs where the initial wear config value was not being loaded correctly. These changes should enhance the overall visual experience of the game. The patch is a result of the upgrade to VS2022, which occurred in preview46.
