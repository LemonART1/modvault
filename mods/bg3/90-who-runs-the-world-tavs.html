About this mod

This mod enables players to create more than one player character, or "Tav," within a single-player Baldur's Gate 3 campaign, facilitating unique "multitav runs." It functions by allowing interaction with specific objects located across Acts 1 and 2, triggering character creation via Script Extender or Osiris scripts, thus avoiding the performance issues of running multiple game instances. Key features include a confirmation pop-up for creation and an autosave function before character generation. The SE version requires the Mod Configuration Menu (MCM) for object management, and players should note specific considerations regarding romances, party limits, and Dark Urge character interactions, which may benefit from companion mods.
