About this mod

This mod overhauls the appearance user interface for Baldur's Gate 3 character creation and the magic mirror. It provides an expanded and rearranged keyboard-only UI, designed to improve the character customization experience. The mod works for both Tav and the guardian, specifically affecting the right-side panels in the character creator. Install like any other .pak mod, preferably placed near the top of your load order for best results. It may conflict with other mods altering character creation xaml files.
