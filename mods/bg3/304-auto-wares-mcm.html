About this mod

The Auto Wares mod for Baldur's Gate 3 allows for automatic marking of items as wares, featuring customization through the MCM menu. This mod is simple, compatible, and lightweight, making it safe to install or uninstall at any time. It includes features for importing and exporting presets, and it can automatically import presets from other mods, such as Auto Loot Seller V4. The load order for this mod is flexible, but it must be below MCM in the load order. The mod also includes a tagging system, with wares tagged as "AUTOWARES" for potential use by other modders.
