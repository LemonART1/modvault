About this mod

This mod provides a robust framework for creating and adding custom companions to Stardew Valley. It empowers mod authors with tools and a detailed wiki on how to develop unique content packs. The mod fully supports multiplayer, allowing custom companions to appear in shared game sessions. It also integrates smoothly with Content Patcher and maintains compatibility with the Save Anywhere mod for enhanced gameplay flexibility.
