About this mod

Farm Type Manager is a Stardew Valley SMAPI framework that enables other mods to spawn objects and monsters anywhere in the world, supports customizable spawn rules such as quantity per day, location, timing, and randomized behavior, works through content packs and integrates automatically with mods using FTM folders, requires SMAPI and Content Patcher installation, supports multiplayer where only host spawn settings apply, and allows global configuration via config.json or Generic Mod Config Menu for in game editing, expanding world activity and modded content variety across gameplay sessions
