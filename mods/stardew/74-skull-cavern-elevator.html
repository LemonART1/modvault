About this mod

Adds an elevator system to Skull Cavern similar to the mine elevator, unlocking new elevator floors every set number of levels, configurable step size, difficulty scaling, first-floor elevator option, and optional travel cost system via config.json, with full SMAPI support and multiplayer compatibility
