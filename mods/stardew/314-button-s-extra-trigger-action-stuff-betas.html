About this mod

Button's Extra Trigger Action Stuff, or BETAS for short, aims to add plenty of new triggers, actions, and game state queries that mod authors can utilize to help their mods "react" to more things or do more useful stuff as a result of things happening. All of the new stuff can be used with any Content Patcher mod so long as this mod is set as a dependency. This mod does nothing on its own. There is no benefit to installing this as a user if another mod that you are using does not require it.
