About this mod

GTA V script that enhances traffic by adding customizable extra vehicles and bikes from a user-defined list without editing base popgroups, supports DLC cars and add-on vehicles, configurable spawn classes like luxury, sports and sedans, region-based spawning for rich, poor and industrial areas, custom colors including chameleon options, vehicle upgrades and liveries with adjustable chances, full control over rarity, performance-friendly system with debug tools and notifications, optional configuration via ini files, install by placing files in scripts folder, requires ScriptHookV Enhanced Edition, ScriptHookVDotNet 3.x, .NET Framework 4.8 and Visual C++ Redistributable 2019 x64
