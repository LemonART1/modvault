About this mod

GTA V script that adds a prison sentence system for single player, sends player to Bolingbroke Penitentiary after death or arrest while wanted, sentence duration based on wanted level, includes prison bus transport, processing sequence, inmate outfits, timer UI, cell entry and exit markers, guard and prisoner AI behavior, restricted area triggers, lockdown system with guard response, baton return to cell mechanic, basic escape system, persistent prison state between sessions, early access feature set with evolving mechanics, install via DLL in scripts folder, requires ScriptHookV, ScriptHookVDotNet 3.6+ and NativeUI plus prison outfit mod
