About this mod

A GUI application for managing UE4SS mods for Subnautica 2. It allows for easy installation, enabling, disabling, and configuration of mods. The manager supports drag-and-drop installation and offers a dark mode UI with a simple, intuitive interface. It also works for other UE4SS games.
