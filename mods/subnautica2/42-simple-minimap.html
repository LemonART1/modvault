About this mod

Simple Minimap adds a lightweight HUD minimap to Subnautica 2 using UE4SS Lua scripting, showing player position, direction, and optional large map view, with toggle controls and fully configurable UI settings like size, opacity, markers, and update interval, supports MapGenie projection calibration, includes fix for Vortex installation bug and fog of war system with configurable grid, reveal radius, and save file, requires UE4SS installation in game Mods folder and supports Win64 and WinGDK versions, optimized for low performance impact with adjustable update rate
