About this mod

This mod allows you to specify a custom number of quick slots for tools and other equippable items. You can configure the exact number by editing the `config.lua` file within the mod's Scripts folder. Please note that this mod overrides the game's normal quick slot progression. Be cautious not to decrease the number of slots if they contain items, as this risks item loss. The mod requires UE4SS and may experience issues with game updates or unpredictable behavior in multiplayer environments.
